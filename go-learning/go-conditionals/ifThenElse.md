<h2 align="center">
An extra else statement can be added to an if condition, to execute what should happen if the if condition is not satisfied.
</h2>

Structure of If Then Else Statement
+---------+
|  Start  | 
+---------+
     |
	 V
	+-------------------+
	| Condition			|-----------+
	+-------------------+			|
		|							|
		| True						| False					
		| 							|
		V							|
	+-----------------+				|
	|  Statements	  |				|
	+-----------------+				|
		|							|
		|							V
		V				   +--------------------+
+---------+				   |					|
|  End	  |<---------------|  Else Statements	|
+---------+				   +--------------------+
